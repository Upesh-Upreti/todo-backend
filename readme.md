
## How to run

To deploy this project 

1.install modules with
```bash
  yarn add
```
or
```bash
  npm install
```
2.Add environmental variables 
```bash
MONGO_USER = 
MONGO_PASSWORD =

HOST=localhost
PORT=3000

#for connecting to mongodb
CONNECTION_STRING = 

JWT_SECRET = 
```
3.Now you can start the server with the follow command
```bash
yarn start
```
